🚨 Cross-Specialty Emergency Screening

▼
Required: Consciously address each emergency for every patient.
Click Ruled Out (evidence against), Monitoring (watching, insufficient data), or Active (being managed) for each condition. Conditions left unclassified will appear as unaddressed in the generated note.
❤️ Cardiovascular
STEMI / Occlusion MI
STE ≥1mm in ≥2 contiguous leads, new LBBB, posterior changes → Cath lab activation
NSTEMI / Unstable Angina
Troponin rise/fall + symptoms or dynamic ECG changes; TIMI / GRACE risk score → early invasive vs. ischemia-guided
Aortic Dissection
Tearing chest/back pain + pulse deficit / BP differential / wide mediastinum → CT aorta; Type A → surgery, Type B → BP control
Cardiac Tamponade
Beck's triad (hypotension, JVD, muffled heart sounds) + pulsus paradoxus >10 mmHg, electrical alternans → urgent pericardiocentesis
Cardiogenic Shock
SBP <90 or MAP <65 + CI <2.2 + PCWP >15 + end-organ hypoperfusion; cold/wet profile → MCS consideration (IABP, Impella)
Hypertensive Emergency
SBP >180 + end-organ damage (encephalopathy, AKI, flash pulm edema, aortic dissection, eclampsia) → IV antihypertensive, 25% MAP reduction in 1h
Malignant Arrhythmia (VT/VF/Complete HB)
Sustained VT → cardioversion if unstable; VF → immediate defibrillation; Complete HB + hemodynamic compromise → transcutaneous pacing
🫁 Pulmonary
Massive / Submassive PE
Massive: hemodynamic instability → systemic tPA or catheter-directed. Submassive: RV strain + troponin elevation → anticoagulation ± PERT activation
Tension Pneumothorax
Respiratory distress + absent breath sounds + tracheal deviation + hemodynamic instability → immediate needle decompression (2nd ICS MCL) before CXR
ARDS / Impending Respiratory Failure
P/F ratio <300 + bilateral infiltrates + non-cardiogenic; SpO2 <90% on ≥10 L O2, RR >30, ↑ WOB → ARDSnet LPV (TV 6 mL/kg IBW, PEEP table)
🧠 Neurological
Acute Ischemic Stroke / LVO
Acute focal deficit → Last known well + NIHSS + CT/CTA. tPA if ≤4.5h + no contraindications; thrombectomy if LVO within 6–24h
Intracranial Hemorrhage / SAH
Thunderclap HA or sudden-onset focal deficit → CT head; if CT negative + high suspicion → LP ≥6h (xanthochromia); CTA for aneurysm
Status Epilepticus
Seizure >5 min or ≥2 without recovery → Lorazepam 0.1 mg/kg IV → Levetiracetam/Fosphenytoin/Valproate → anesthesia if refractory
Increased ICP / Herniation
Cushing triad (HTN + bradycardia + irregular RR) or posturing → HOB 30°, head midline, mannitol 1 g/kg or 23.4% NaCl; neurosurgery immediately
Wernicke Encephalopathy
Any malnourished/ETOH patient with ≥1 of: ataxia, ophthalmoplegia, confusion → Thiamine 500 mg IV TID ×3 days BEFORE any dextrose
🫘 Renal & Metabolic
Dialysis Emergency (AEIOU)
Acidemia pH <7.1 | Electrolytes (K>6.5 or refractory) | Intoxication (dialyzable) | Overload (refractory pulm edema) | Uremia (encephalopathy, pericarditis) → emergent RRT
Life-Threatening Hyperkalemia
K >6.0 or any K with ECG changes (peaked T, wide QRS, sine wave) → Calcium gluconate IV → Insulin/D50 → Kayexalate/patiromer → RRT if refractory
Severe Symptomatic Hyponatremia / Hypernatremia
Na <125 + AMS/seizure → 3% NaCl 150 mL bolus; target +4–6 mEq/L in 6h. Na >155 + AMS → free water deficit, correct ≤10 mEq/day to prevent cerebral edema
🦋 Endocrine
DKA / HHS
DKA: BG >250 + AG acidosis + ketonemia. HHS: BG >600 + Osm >320 + altered MS without significant acidosis → IV fluids, insulin drip (DKA), K repletion, hourly monitoring
Adrenal Crisis
Hypotension + hyponatremia + hyperkalemia + cortisol <3 (or steroid non-adherence) → Hydrocortisone 100 mg IV bolus NOW; then 50–100 mg q6–8h + aggressive IVF
Thyroid Storm / Myxedema Coma
Storm (Burch-Wartofsky ≥45): PTU/methimazole → iodine (1h after) → steroids → beta-blocker. Myxedema: IV T4 + hydrocortisone (cover adrenals first) + rewarm
Severe Hypercalcemia / Hypocalcemia
Ca >14: aggressive IVF (200–300 mL/h) + calcitonin 4 IU/kg q12h + bisphosphonate. Ca critically low with tetany/ECG changes: Calcium gluconate 1–2 g IV over 20 min
🦠 Infectious Disease & Sepsis
Septic Shock (Sepsis-3)
Sepsis + vasopressor requirement + lactate >2 mmol/L → 30 mL/kg IVF in 3h, blood cultures ×2 before, broad-spectrum Abx within 1h; norepinephrine first-line pressor
Bacterial Meningitis / Encephalitis
Fever + AMS + meningismus → Dexamethasone + Ceftriaxone + Vancomycin (± Ampicillin if >50y) immediately; LP can follow if no focal deficit or papilledema; add Acyclovir if HSV encephalitis possible
Infective Endocarditis
Fever + new murmur + bacteremia (Staph aureus, Strep, Enterococcus) → Modified Duke Criteria; TTE/TEE; 3 sets blood cultures pre-Abx; urgent cardiology if hemodynamic instability or emboli
Necrotizing Soft Tissue Infection
Rapidly advancing erythema + skin necrosis + crepitus + disproportionate pain or pain out of proportion → Emergent surgical debridement + Pip-tazo + Vanc + Clindamycin (anti-toxin)
🩸 Hematologic
TTP / HUS
Thrombocytopenia + microangiopathic hemolytic anemia (schistocytes) + renal failure ± neuro/fever → ADAMTS13 activity; plasma exchange immediately; avoid platelets in TTP
DIC
↓Plt + ↑PT/PTT + ↑D-dimer + ↓fibrinogen + schistocytes in sepsis/malignancy/trauma → treat underlying cause; FFP if bleeding, cryoprecipitate if fibrinogen <150
Tumor Lysis Syndrome
Cairo-Bishop criteria: ↑K, ↑Phos, ↑Uric acid, ↓Ca, ↑Cr in 24–48h after chemo or spontaneously in high-burden malignancy → aggressive IVF + allopurinol/rasburicase + monitor electrolytes q6h
🦴 Rheumatologic / Other
Macrophage Activation Syndrome (MAS) / HLH
Persistent fever + cytopenias + ferritin >10,000 + hepatosplenomegaly + ↑TG + ↓fibrinogen + hemophagocytosis on BM bx → HScore; dexamethasone ± etoposide; urgent rheumatology/heme
Life-Threatening GI Hemorrhage
Hematemesis or melena + hemodynamic instability (SBP <90, HR >100) → 2 large-bore IVs, type & cross, PPI IV drip, GI + IR on call; transfuse to Hgb >7–8 g/dL; massive transfusion protocol if needed
Airway Emergency / Impending Arrest
Stridor, angioedema, inability to protect airway, SpO2 <88% refractory to supplemental O2 → Early intubation; anaphylaxis → Epinephrine 0.3 mg IM immediately
